ΔΩΡΕΑΝ ΜΑΘΗΜΑΤΑ JAVA

Στο σημερινό δωρεάν μάθημα java, θα αναφερθούμε σε μια καινούργια ιδιότητα που έχει προστεθεί από την έκδοση 10 και μετά που μας δίνει την ικανότητα να ορίσουμε μεταβλητές χωρίς να ορίζουμε εξ αρχής το data type. Το local variable type inference όπως ονομάζεται μας προσφέρει πολλά πλεονεκτήματα όπως θα δούμε στην συνέχεια. Αλλά ας τα πάρουμε τα πράγματα από την αρχή.

Όπως ήδη γνωρίζουμε, στην java πρέπει να ορίσουμε εξ αρχής το data type της μεταβλητής και μετά να αναθέσουμε μια τιμή στην μεταβλητή η οποία συμφωνεί με το προσδοκώμενο data type. Με την καινούργια ιδιότητα που προστέθηκε στην java, το data type θα το πάρει η μεταβλητή από το είδος της τιμής που θα της αναθέσουμε.

Για να ορίσουμε μια local variable type inference χρησιμοποιούμε το λέξη κλειδί var και αμέσως μετά ακολουθεί το όνομα της μεταβλητής με την τιμή της.

var number_students = 10;

Με βάση την θεωρία, από την στιγμή που αναθέτουμε μια int τιμή στην μεταβλητή number_students, τότε αυτή η μεταβλητή είναι είδος int. Αν τώρα χρησιμοποιήσετε το όνομα var για όνομα μιας δικής σας μεταβλητής τότε αυτό είναι ένα απλό όνομα και δεν έχει καμία έξτρα ιδιότητα. Είναι σαν να ορίζεται μια κοινή μεταβλητή όπως ήδη γνωρίζουμε.

int var = 1;

Ας δούμε ένα απλό παράδειγμα. Δημιουργήστε ένα καινούργιο project με το όνομα VarDemo και μέσα σε αυτό ένα πακέτο με το όνομα com.mycompany. Μέσα στο πακέτο δημιουργήστε μια καινούργια κλάση με το όνομα VarDemo και γράψε τον εξής κώδικα:

VarDemo.java

    package com.mycompany;

public class VarDemo {
    public static void main(String[] args) {
        var num = 5.0;
        System.out.println("Value of num: " + num);
        
        int var = 1;
        System.out.println("Value of var: " + var);
        
    }
}

Output

    Value of num: 5.0
    Value of var: 1

Με τον ίδιο ακριβώς τρόπο μπορούμε και να ορίσουμε έναν array. Προσέχουμε όμως να μην τοποθετήσουμε brackets ( [ ] ) στο όνομα του array.

var students = new int[20];

Αν και μπορεί να φαίνεται εύχρηστη και βολική αυτή η καινούργια ιδιότητα, υπάρχει ένας περιορισμός – μπορεί μόνο να χρησιμοποιηθεί για για να ονομάσουμε τοπικές μεταβλητές (local variables). Δεν μπορεί να χρησιμοποιηθεί για να ορίσουμε μεταβλητές μιας κλάσης (θα μάθουμε για τις κλάσεις αργότερα). Επίσης πρέπει να γράψετε στην ίδια γραμμή κώδικα και την var μεταβλητή και την τιμή της.

var students; //Λάθος

students = 10; //Λάθος
