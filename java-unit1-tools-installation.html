ΕΝΟΤΗΤΑ 1 - Εγκατάσταση εργαλείων για ανάπτυξη προγραμμάτων Java

Michail Kassapoglou
12
Oct

Το πρώτο βήμα που πρέπει να ολοκληρώσουμε, πριν ακόμα γράψουμε τον πιο απλό κώδικα σε Java, είναι να εγκαταστήσουμε τα απαραίτητα εργαλεία που θα μας βοηθήσουν να γράψουμε, να κάνουμε διορθώσεις και να εκτελέσουμε την τελική έκδοση του κώδικα μας. Αν και έχουμε πολλές επιλογές εργαλείων, το θετικό στοιχείο είναι ότι όλα τα εργαλεία είναι αξιόπιστα ως προς σταθερότητα τους σαν προγράμματα και δωρεάν να τα χρησιμοποιήσουμε ακόμα και για επαγγελματικές εφαρμογές που θα μπορούσαν να πουληθούν.

Τα βασικά ερωτήματα είναι δύο: (1) τι εργαλεία χρειαζόμαστε και (2) ποια είναι η σωστή σειρά εγκατάστασή τους.

Για την ανάπτυξη προγραμμάτων σε γλώσσα προγραμματισμού Java, θα χρειαστεί να έχουμε τα εξής εργαλεία διαθέσιμα και εγκαταστημένα στον υπολογιστή μας:

  • Το Java Development Kit (JDK) από την Oracle
  • Το maven από το Apache Maven Project
  • Ένα Integrated Development Enviroment (IDE) για την δημιουργία και εκτέλεση Java κώδικα.

Ξεκινώντας από την Java, για να μπορέσουμε να βρούμε την πιο τελευταία έκδοση της θα πρέπει να πάμε στο site της Oracle (https://www.oracle.com/technetwork/java/javase/downloads/index.html). Στην αρχή η λίστα μπορεί να σας μπερδέψει οπότε ας εξηγήσουμε λίγο τις επιλογές που υπάρχουν στο site. Η πιο τελευταία έκδοση βρίσκεται στην αρχή της λίστας. Το κουμπί δεξιά μας δίνει μια και μόνο επιλογή για να κατεβάσουμε που είναι το Oracle JDK. Αυτό είναι το περιβάλλον που χρειαζόμαστε για να γράψουμε και ταυτόχρονα να τρέξουμε έναν πρόγραμμα γραμμένο σε java. Αν κατεβούμε πιο κάτω στην λίστα θα δούμε παλαιότερες εκδόσεις που εκτός από την JDK επιλογή έχουν και την JRE επιλογή. Μετά την έκδοση Java 10, η JRE επιλογή δεν υπάρχει πια όπως δεν υπάρχει πια και η επιλογή για 32 ή 64 bit εγκατάσταση. Όλες οι καινούργιες εκδόσεις της Java απαιτούν να τρέξουν μόνο σε 64 bit περιβάλλον. Τέλος αν θέλετε να αναπτύξετε κάποιο πρόγραμμα που αργότερα θα πουλήσετε και δεν θέλετε να πληρώσετε license fees για την χρήση της Java σε παραγωγικό περιβάλλον μπορείτε να κατεβάσετε την open source έκδοση (jdk.java.net).

Οπότε για να ανακεφαλαιώσουμε, κατεβάστε την πιο τελευταία έκδοση από την Oracle και εγκαταστήσετε την. Απλά δεχτείτε όλες τις προτεινόμενες επιλογές. Τα δικά μας παραδείγματα θα γραφτούν σε περιβάλλον Windows 10 με Java 13. Εδώ θα ήθελα να επισημάνω το γεγονός ότι αν και η έκδοση της Java έχει φτάσει στην έκδοση 13 (μέχρι την στιγμή που γράφεται αυτό το άρθρο), πολλές εταιρείες έχουν μείνει και συνεχίζουν να γράφουν σε Java 8. Η θεωρία σε ένα μεγάλο ποσοστό είναι η ίδια, θα αναφέρουμε όμως τις διαφορές που υπάρχουν ανάμεσα στην Java 8 και στις υπόλοιπες εκδόσεις μετά από αυτήν.
