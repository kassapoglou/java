ΔΩΡΕΑΝ ΜΑΘΗΜΑΤΑ JAVA

Στο σημερινό δωρεάν μάθημα Java θα ασχοληθούμε με τους arrays. Θα δούμε τους ποιο συχνούς τρόπους για να ορίσουμε σωστά έναν array ενώ ταυτόχρονα θα δούμε πολλά αλλά απλά παραδείγματα για να κατανοήσουμε καλύτερα την θεωρία. Τι είναι όμως ένας array?

Για να μπορέσουμε να κατανοήσουμε καλύτερα το πως μας διευκολύνει ένας array, ας αναλύσουμε ένα θεωρητικό παράδειγμα. Σκεφτείτε λοιπόν ότι μας έχουν αναθέσει την ανάπτυξη ενός java προγράμματος που σκοπός του είναι η καταγραφή των μαθητών. Υπάρχουν αρκετοί τρόποι να προσεγγίσετε αυτό το πρόβλημα και να το λύσετε, αλλά με βάση αυτά που γνωρίζουμε μέχρι τώρα από τις προηγούμενες ενότητες η απάντηση δεν είναι και τόσο εύκολη. Ας πούμε ότι θεωρείτε ότι κάθε τάξη δεν μπορεί να έχει περισσότερους από 20 μαθητές, οπότε αποφασίζετε να δημιουργήσετε 20 διαφορετικές μεταβλητές που κάθε μια θα αντιστοιχεί σε έναν μαθητή. Και αν η τάξη είχε 50 μαθητές τότε θα έπρεπε να δημιουργήσετε 50 διαφορετικές μεταβλητές. Αν νομίζετε ότι μπορείτε να διαχειριστείτε αυτό τον αριθμό των μεταβλητών, τότε πως θα αντιμετωπίζατε το πελατολόγιο μιας τράπεζας? Θα γράφατε 5 ή ακόμα 50 χιλιάδες μεταβλητές? Δεν νομίζω. Αλλά αν ακόμα τις γράφατε η κατανάλωση μνήμης του υπολογιστή στον οποίο θα έτρεχε το πρόγραμμα σας θα ήταν πολύ μεγάλη. Εδώ λοιπόν είναι που χρειαζόμαστε την λογική του array.

Σαν array ονομάζουμε μια συλλογή στοιχείων, σε συνεχόμενες διευθύνσεις στην μνήμη, του ίδιου τύπου δεδομένων που αντιπροσωπεύονται από ένα μόνο όνομα. Κάθε στοιχείο του array πρέπει να είναι ενός συγκεκριμένου, και εξ αρχής ορισμένου, είδος δεδομένων. Το ένα στοιχείο από το άλλο ξεχωρίζει από την θέση που κατέχει μέσα στον array και χαρακτηρίζεται από έναν αριθμό που ονομάζεται index. Η αρίθμηση του index ξεκινάει από το 0. Στην ουσία για να δημιουργήσουμε έναν array πρέπει να ακολουθήσουμε δύο βήματα:

1. Να δώσουμε ένα όνομα στον array

2. Να δημιουργήσουμε τον array χρησιμοποιώντας την λέξη κλειδί new και ορίζοντας το μέγεθος του

Στην java, τα δεδομένα θα ανήκουν είτε στα primitive types ή θα είναι είναι αντικείμενα. Οι arrays είναι αντικείμενα για αυτό και δεν δημιουργούμε στην ουσία μεταβλητή αλλά reference για να μπορέσουμε να τους διαχειριστούμε. Μην ανησυχείτε αν δεν καταλαβαίνετε την έννοια του αντικειμένου. Ακόμα δεν έχουμε μιλήσει για αυτά. Για τώρα απλά δεχτείτε το συντακτικό δημιουργίας ενός array όπως έχει.

Πως λοιπόν ορίζουμε έναν array? Πρώτα δηλώνουμε το είδος των δεδομένων που θα δέχεται ο array όπως int, float, string κτλ. Αμέσως μετά γράφουμε τα square brackets σύμβολα [ ] με το όνομα του array. Μην ξεχάσατε να κλείσετε το statement με semicolon ( ; ) στο τέλος.

int [ ]students;

Η java μας επιτρέπει να γράφουμε τα brackets μετά από το όνομα σαν δεύτερη εναλλακτική δήλωση του array. Και οι δύο δηλώσεις είναι σωστές και οι δύο χρησιμοποιούνται συχνά.

int students[ ];

Μέχρι τώρα έχουμε απλά ορίσει το reference του array με το όνομα students. Αλλά μας λείπει ακόμα να ορίσουμε το μέγεθος του πριν μπορέσουμε να το χρησιμοποιήσουμε. Σαν δεύτερη δήλωση λοιπόν ορίζουμε το μέγεθος του array με την λέξη κλειδί new και με τον αριθμό των δεδομένων που μπορεί να δέχεται. Ας ορίσουμε λοιπόν έναν array που θα δέχεται 20 μαθητές.

students = new int[20];

Εννοείται ότι έχουμε την δυνατότητα να ορίσουμε το reference (το όνομα του array) όπως και το μέγεθος του στην ίδια γραμμή κώδικα.

int students[] = new int[20];

Η παραπάνω εντολή δημιουργεί έναν array που δέχεται 20 ακέραιους αριθμούς. Αυτό που απομένει τώρα είναι να μάθουμε πως μπορούμε να τοποθετήσουμε τους αριθμούς μας σε κάθε μια από αυτές τις 20 θέσεις. Ευτυχώς ο array μας προσφέρει έναν απλό τρόπο για αυτό και ονομάζεται index. Ο array εσωτερικά δίνει έναν αριθμό σε κάθε μια από τις θέσεις του ξεκινώντας την αρίθμηση από το 0. Οπότε για έναν array που δέχεται 20 ακέραιους αριθμούς σαν δεδομένα, η αρίθμηση του index θα είναι από 0 έως 19. Πάντα θα υπολογίζουμε -1 από τον ολικό αριθμό των δεδομένων που δέχεται ο array.

Αν θέλαμε λοιπόν να βάλουμε την βαθμολογία του πρώτου μαθητή στον array, θα πρέπει να γράψουμε το εξής

students[0] = 78;

Ενώ για τον δεύτερο μαθητή θα μπορούσαμε να βάλουμε την βαθμολογία στην αμέσως επόμενη θέση που είναι

students[1] = 99;

Στην ουσία έχουμε δημιουργήσει αυτό

Μια ερώτηση που συχνά γεννιέται όταν βλέπουμε ένα τέτοιο παράδειγμα είναι τι τιμές έχουν οι υπόλοιπες θέσεις του array στις οποίες ακόμα δεν έχουμε βάλει κάποια τιμή. Ένας array δεν είναι primitive type (όπως π.χ int, float, char) αλλά είναι αντικείμενο. Για αυτό άλλωστε και χρησιμοποιούμε την λέξη κλειδί new κατά την δημιουργία του. Ακόμα δεν έχουμε μιλήσει για αντικείμενα και ούτε χρειάζεται να μπούμε σε περισσότερες λεπτομέρειες. Όμως, λόγω της ιδιότητας του, ο array για όσες θέσεις του δεν έχει δεχτεί από εμάς κάποια τιμή, θα τις αρχικοποιήσει με την τιμή 0. Αν τώρα αντί για int είχαμε ορίσει τον array να δέχεται double τιμές τότε ο array θα τοποθετούσε 0.0 στις θέσεις εκείνες που δεν έχουν τιμές από εμάς.

Αν πάλι επιθυμούσατε να γεμίσετε τον array με κάποια άλλη αρχική τιμή εκτός από το 0 θα μπορούσατε να χρησιμοποιήσετε ένα for loop για να γεμίσετε τον array γρήγορα με την τιμή που θέλετε.

Και πως διαβάζουμε τις τιμές από έναν array? Ακριβώς με τον ίδιο τρόπο με τον οποίο εισάγατε τις τιμές με βάση το index. Και πάλι θα μπορούσατε να χρησιμοποιήσετε ένα for loop για να διαβάσετε γρήγορα τις τιμές που περιέχει. Ας δούμε ένα απλό παράδειγμα που μας δείχνει όλα όσα μάθαμε μέχρι τώρα.

Δημιουργούμε ένα project με το όνομα BasicArray. Έπειτα δημιουργούμε ένα πακέτο με το όνομα com.mycompany και μέσα στον πακέτο δημιουργούμε μια κλάση με το όνομα BasicArray.

BasicArray.java

    package com.mycompany;

    public class BasicArray {
        public static void main(String[] args) {
            int students[] = new int[5];
            System.out.println("####### Array without input values #######");
            
            for (int i = 0; i < 5; i++) {
                System.out.println("Current value of index "+i+" is "+students[i]);
            }
            
            students[2]=78;
            students[4]=99;
            
            System.out.println("####### Array with input values #######");
            
            for (int i = 0; i < 5; i++) {
                System.out.println("Current value of index "+i+" is "+students[i]);
            }
        }
    }

Output

    Current value of index 4 is 0
    ####### Array with input values #######

    Current value of index 1 is 0

    Current value of index 3 is 0

Το παράδειγμα μας ήταν πολύ απλό. Όριζε εξ αρχής έναν array που θα μπορούσε να δεχτεί μέχρι και 5 ακέραιους αριθμούς. Αμέσως μετά ζητάμε με ένα for loop να δούμε τις default τιμές που έχει δώσει η java στον array. Μετά ερχόμαστε να βάλουμε εμείς τις δικές μας τιμές. Δεν είναι απαραίτητο να βάζετε τις τιμές με την σειρά στον array. Μπορείτε να τις τοποθετήσετε όπου θέλετε μέσα στον array. Αυτό άλλωστε δείχνει και ο κώδικας μας. Και τέλος ερχόμαστε να εκτυπώσουμε τις τιμές που περιέχει ο array για άλλη μια φορά για να δούμε αν όντως οι τιμές είναι αποθηκευμένες στον array. Προσέξτε λίγο στην συνθήκη του for loop. Έχουμε ορίσει να είναι μικρότερη του 5 γιατί το index ξεκινάει από το μηδέν και στο συγκεκριμένο παράδειγμα φτάνει μέχρι το 4. Αν προσπαθήσετε να βάλετε τιμή ή να διαβάσετε τιμή από έναν index που δεν υπάρχει (π.χ students[5]) τότε θα λάβετε το μήνυμα λάθους ArrayIndexOutOfBounds από το runtime περιβάλλον της java. Ας δούμε ένα απλό παράδειγμα. Δημιουργείστε μια καινούργια κλάση με το όνομα ArrayError και εκτελέστε τον παρακάτω κώδικα.

ArrayError.java

    package com.mycompany;

    public class ArrayError {
        public static void main(String[] args) {
            int students[] = new int[5];
           
            students[2]=78;
            students[4]=99;
            students[5]=63;
            
        }
    }

Output

    Exception in thread "main" java.lang.ArrayIndexOutOfBoundsException: Index 5 out 
    of bounds for length 5 at com.mycompany.ArrayError.main(BasicArray.java:10)
Ένα μέρος στο οποίο συχνά κάνουμε το λάθος να ζητάμε ένα index που δεν υπάρχει στον array είναι το for loop με το οποίο διαβάζουμε τις τιμές του array. Για αυτό ακριβώς το λόγο πρέπει να προσέχουμε πολύ με την συνθήκη την οποία ορίζουμε. Όμως επειδή ένας array είναι αντικείμενο, η java έχει φροντίσει να το επενδύσει με αρκετές ιδιότητες οι οποίες μπορούν να μας βοηθήσουμε να αποφύγουμε λογικά λάθη. Μια τέτοια ιδιότητα (ή attribute όπως ονομάζεται) είναι η length. Αντί λοιπόν να γράφουμε i < 5 μπορούμε να γράψουμε students.length αποφεύγοντας τυχόν ArrayIndexOutOfBounds λάθη. Ας δούμε πως μπορούμε να γράψουμε το προηγούμενο πρόγραμμα μας χρησιμοποιώντας την length attribute.

BasicArray.java

    package com.mycompany;

    public class BasicArray {
        public static void main(String[] args) {
            int students[] = new int[5];
            System.out.println("####### Array without input values #######");
            
            for (int i = 0; i < students.length; i++) {
                System.out.println("Current value of index "+i+" is "+students[i]);
            }
            
            students[2]=78;
            students[4]=99;
            
            System.out.println("####### Array with input values #######");
            
            for (int i = 0; i < students.length; i++) {
                System.out.println("Current value of index "+i+" is "+students[i]);
            }
        }
    }

Η java μας επιτρέπει, εφόσον έχουμε μόνο λίγα δεδομένα να περάσουμε στον array, να ορίσουμε τον array και να του δώσουμε τις τιμές στην ίδια γραμμή κώδικα και χωρίς μάλιστα να χρησιμοποιήσουμε την λέξη κλειδί new. Αυτός είναι ένας εύκολος και σύντομος τρόπος αν έχουμε έναν μικρό αριθμό δεδομένων. Ο τρόπος που ορίζουμε έναν τέτοιο array είναι ο εξής:

int [ ]students = { 67, 49, 72, 83, 99};

Ορίζουμε έναν array με το όνομα του και το είδος των δεδομένων που δέχεται. Μέσα σε άγκιστρα δηλώνουμε τις τιμές που θα περάσουμε στον array. Ο τρόπος όπως βλέπετε είναι απλός αλλά πρέπει να γίνει η δήλωση του ονόματος του array και η ανάθεση τιμών στην ίδια γραμμή κώδικα. Ας δούμε ένα ολοκληρωμένο πρόγραμμα. Δημιουργείστε μια καινούργια κλάση με το όνομα ArrayInit και γράψτε τον παρακάτω κώδικα:

ArrayInit.java

    package com.mycompany;

    public class ArrayInit {
    
        public static void main(String[] args) {
            int[] odds = {1, 3, 5, 7, 9};
            System.out.println("odds.length = " + odds.length);
    
            for (int i = 0; i < odds.length; i++) {
                System.out.println("odds[" + i + "] = " + odds[i]);
            }
    
            String[] daysOfWeek = {"Saturday", "Sunday", "Monday","Tuesday", 
                "Wednesday", "Thursday", "Friday"};
            System.out.println ("daysOfWeek.length = " + daysOfWeek.length);
        
            for(int i = 0;i< daysOfWeek.length ;i++)
            {
                System.out.println("daysOfWeek[" + i + "] = "+ daysOfWeek[i]);
            }
    
        }
    }

Εκτός από το for loop έχουμε έναν ακόμα πιο γρήγορο τρόπο για διαβάζουμε τις τιμές από έναν array και ονομάζεται for-each loop. Η δομή του είναι πολύ απλή. Δηλώνουμε μια τοπική μεταβλητή στο for-each loop με ότι όνομα επιθυμούμε και ορίζουμε πιο είναι το data type της μεταβλητής. Μετά χρησιμοποιούμε το colon ( : ) και αμέσως μετά ακολουθεί το όνομα του array. Ας δούμε ένα απλό παράδειγμα. Δημιουργείστε μια καινούργια κλάση με το όνομα ForEach και γράψτε τον παρακάτω κώδικα:

ForEach.java

    package com.mycompany;

public class ForEach {

    public static void main(String[] args) {

        String[] daysOfWeek = {"Saturday", "Sunday", "Monday","Tuesday", 
            "Wednesday", "Thursday", "Friday"};

    
        for(String days : daysOfWeek)
        {
            System.out.println(days);
        }

    }
}

Output

    Saturday
    Sunday
    Monday
    Tuesday
    Wednesday
    Thursday
    Friday

Όλα τα παραδείγματα που έχουμε αναλύσει μέχρι τώρα περιέχουν arrays μιας διάστασης. Πως θα μπορούσαμε να ορίσουμε έναν array δύο ή και περισσοτέρων διαστάσεων? Ο τρόπος ξανά είναι απλός. Προσθέτουμε τόσα brackets ( [ ] ) όσες είναι οι διαστάσεις που θέλουμε να έχει ο array. Ας δούμε πως ορίζεται ένας array με δύο διαστάσεις:

int [ ] [ ]courses = new int[3] [2];

Στην ουσία έχουμε δημιουργήσει αυτό:

Ας δούμε ένα απλό παράδειγμα στο οποίο ορίζουμε έναν array με δύο διαστάσεις και προσθέτουμε μερικές τιμές. Δημιουργείστε μια καινούργια κλάση με το όνομα ArrayDim και γράψτε τον παρακάτω κώδικα:

ArrayDim.java

    package com.mycompany;

public class ArrayDim {

    public static void main(String[] args) {
        int[][] courses = new int[3][2];
        courses[1][1] = 10;
        courses[2][0] = 20;

        for (int row = 0; row < courses.length; row++) {
            for (int col = 0; col < courses[row].length; col++) {
                System.out.println(courses[row][col]);
}
System.out.println();
        }
    }
}

Output

    0
    0
    
    0
    10
    
    20
    0

Οι array είναι πολύ χρήσιμοι στην καθημερινότητα μας σαν προγραμματιστές όμως παρουσιάζουν μια μεγάλη αδυναμία – το μέγεθος τους δεν μπορεί να αλλάξει εφόσον οριστεί. Αυτό είναι ένα μεγάλο μειονέκτημα. Σκεφτείτε πάλι το παράδειγμα με τους πελάτες μια τράπεζας. Πόσο μεγάλο θα ορίσετε εξ αρχής τον array σας? 10,000 πελάτες, 500,000 πελάτες? Και αν σε ένα χρόνο ή δύο διπλασιαστούν οι πελάτες θα πρέπει να αλλάξετε τον κώδικα? Και αν δεν διπλασιαστούν, τότε θα έχετε δεσμεύσει αρκετή μνήμη από το σύστημα χωρίς να χρησιμοποιείται. Εκτός από αυτό το πρόβλημα υπάρχουν και άλλα προβλήματα όπως ταυτόχρονη πρόσβαση στα δεδομένα από διαφορετικά αντικείμενα και προβλήματα απόδοσης και ταχύτητας. Οπότε κάποια στιγμή αργότερα, όταν μάθουμε την έννοια των αντικειμένων, θα μπορέσουμε να μιλήσουμε για τα collections και πως αυτά έχουν λύσει τα προβλήματα που αντιμετωπίζουμε με τον array σε μεγάλο όγκο δεδομένων.
