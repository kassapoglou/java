ΕΝΟΤΗΤΑ 9 JAVA SE - WHILE, DO-WHILE AND FOR LOOP

Στο σημερινό δωρεάν μάθημα Java, θα δούμε τα loops, δηλαδή τον τρόπο με τον οποίο μπορούμε να τρέξουμε πολλαπλές φορές μια λογική του προγράμματος μας γράφοντας τον κώδικα μια μόνο φορά. Η Java μας προσφέρει τρία διαφορετικά είδη loops τα οποία και θα τα αναλύσουμε όλα σε αυτή την ενότητα: while loop, do-while loop και for loop. Ας ξεκινήσουμε την ανάλυση μας με το while loop.

Το while loop έχει την δυνατότητα να εκτελεί ένα κομμάτι κώδικα συνέχεια όσο η συνθήκη η οποία ελέγχει το while loop είναι true. Αν η συνθήκη είναι false τότε το while loop σταματάει την εκτέλεση του και το πρόγραμμα προχωράει με την εκτέλεση του κώδικα που ακολουθεί μετά από το while loop.

Η γενική μορφή που ορίζει το while loop είναι:

while(Boolean_expression)

{

//Statements

}

Η αρχή ενός while loop ορίζεται από την λέξη κλειδί while. Αμέσως μετά ακολουθούν ένα ζεύγος παρενθέσεων που περιέχουν την συνθήκη. Το αποτέλεσμα της συνθήκης πρέπει να επιστρέφει true ή false. Όσο η συνθήκη επιστρέφει true τότε το while loop θα εκτελεί τον κώδικα που ανήκει στο loop. Εάν η συνθήκη γίνει false τότε θα παραβλέψει το κομμάτι κώδικα που ανήκει στο while loop και θα συνεχίσει με το υπόλοιπο πρόγραμμα.

Ας δούμε ένα απλό while loop πρόγραμμα πριν μιλήσουμε για ακόμα μια σημαντική λεπτομέρεια που αφορά το while loop. Δημιουργούμε ένα καινούργιο project με το όνομα WhileDemo. Μέσα στο project δημιουργούμε ένα πακέτο με το όνομα com.mycompany και μέσα στο πακέτο μια κλάση με το όνομα WhileDemo. Ο κώδικας της κλάσης είναι ο ακόλουθος:

WhileDemo.java

											
    package com.mycompany;

    public class WhileDemo {

        public static void main(String[] ags) {
            int x = 1;
            while (x <= 10) {
                System.out.println(x);
                x++;
            }
        }
    }

Αυτό που πρέπει να παρατηρήσουμε σε αυτό το απλό πρόγραμμα είναι ότι μια από τις γραμμές κώδικα που εκτελεί το while loop είναι και η αύξηση της int x κατά 1. Αυτό σημαίνει ότι επηρεάζουμε την τιμή της μεταβλητής μέσα από το loop έτσι ώστε κάποια στιγμή η συνθήκη του while loop να είναι false και να ολοκληρωθεί η εκτέλεση ο κύκλος των επαναλήψεων. Εάν δεν φροντίσετε να έχετε κάποιο κώδικα μέσα στο loop που να επηρεάζει την συνθήκη τότε υπάρχει κίνδυνος να καταλήξετε με ένα loop που δεν σταματάει ποτέ και ονομάζεται infinite loop.

Ο παρακάτω κώδικας είναι ένα δείγμα ενός infinite loop

int i = 1;

while (i > 0)

{

System.out.println(i++);

}

Το do-while loop συμπεριφέρεται σχεδόν σαν το while loop αλλά διαφέρει στο πότε ελέγχει την συνθήκη. Ενώ το while loop πρώτα κάνει τον έλεγχο της συνθήκης, το do-while κάνει τον έλεγχο στο τέλος αφού έχει ήδη εκτελέσει τον κώδικα που ανήκει στο loop μια φορά. Άλλωστε αυτός είναι και ο λόγος που έχουμε δύο loops που σχεδόν συμπεριφέρονται το ίδιο. Το while loop, αν η συνθήκη είναι εξ αρχής false, δεν πρόκειται να εκτελεστεί. Το do-while όμως αφήνει τον κώδικα να τρέξει στα σίγουρα μια φορά. Αν τώρα στο τέλος της εκτέλεσης η συνθήκη επιστρέφει true τότε θα εκτελέσει ξανά την λογική που ανήκει στο do-while loop. Αν όμως είναι false τότε δεν θα εκτελεστεί ξανά.

Η γενική δομή ενός do-while loop είναι η εξής:

do

{

//Statements

}while(Boolean_expression);

Ας δούμε ένα απλό παράδειγμα ενός do-while loop. Δημιουργούμε ένα καινούργιο project με το όνομα DoWhileDemo. Μέσα στο project δημιουργούμε ένα πακέτο με το όνομα com.mycompany και μέσα στο πακέτο μια κλάση με το όνομα DoWhileDemo. Ο κώδικας της κλάσης είναι ο ακόλουθος:

DoWhileDemo.java

											
    package com.mycompany;

public class DoWhileDemo {
    public static void main(String []args){
        int y = 10;
        do
        {
            System.out.println(y);
            y += 10;
        }while(y <= 100);
    }
}

Τα δύο προηγούμενα loops που ήδη αναλύσαμε, επαναλαμβάνουν την εκτέλεση ενός συγκεκριμένου κώδικα για όσο η συνθήκη που τα καθορίζει είναι true. Αυτό σημαίνει ότι δεν στηριζόμαστε στον ακριβή αριθμό των επαναλήψεων γιατί μπορεί να μην το γνωρίζουμε. Το τρίτο και τελευταίο loop που ονομάζεται for loop έρχεται να καλύψει αυτό ακριβώς το κενό, δηλαδή να εκτελέσει ένα κομμάτι κώδικα για ένα συγκεκριμένο αριθμό επαναλήψεων που είναι γνωστός από την αρχή.

Η γενική δομή ενός for loop ορίζεται έτσι:

for(initialization; Boolean_expression; update)

{

//Statements

}

Πριν δούμε ένα απλό παράδειγμα με το for loop, ας δούμε τι ακριβώς σημαίνουν οι παράμετροι που πρέπει να ορίσουμε.

Initialization – Εδώ ορίζουμε την μεταβλητή ή μεταβλητές που θα χρησιμοποιηθούν από το for loop.

Boolean expression – Είναι η συνθήκη που όσο παραμένει true το for loop θα συνεχίζει να εκτελείται

update – Εδώ ορίζουμε την αριθμητική πράξη που θα επηρεάζει την μεταβλητή που ορίσαμε στο initialization.

Ας δούμε ένα απλό παράδειγμα ενός for loop. Δημιουργούμε ένα καινούργιο project με το όνομα ForDemo. Μέσα στο project δημιουργούμε ένα πακέτο με το όνομα com.mycompany και μέσα στο πακέτο μια κλάση με το όνομα ForDemo. Ο κώδικας της κλάσης είναι ο ακόλουθος:

ForDemo.java

											
    package com.mycompany;

public class ForDemo {

    public static void main(String[] args) {
        for (int j = 1; j <= 1024; j = j * 2) {
            System.out.println(j);
        }
    }
}

Είναι πολύ συνηθισμένο φαινόμενο να δείτε δύο ακόμα λέξεις κλειδιά να χρησιμοποιούνται σε συνδυασμό με όλα τα loops στα οποία έχουμε αναφερθεί – το break και το continue.

Το break το έχουμε ήδη συναντήσει στο switch. Στα loops έχει βασικά την ίδια ακριβώς λειτουργία – να σταματάει το loop ανά πάσα στιγμή και να βγαίνει έξω από την λειτουργία του loop συνεχίζοντας με το υπόλοιπο πρόγραμμα.

Ας δούμε ένα απλό παράδειγμα ενός while loop που χρησιμοποιεί το break. Δημιουργούμε ένα καινούργιο project με το όνομα BreakDemo. Μέσα στο project δημιουργούμε ένα πακέτο με το όνομα com.mycompany και μέσα στο πακέτο μια κλάση με το όνομα BreakDemo. Ο κώδικας της κλάσης είναι ο ακόλουθος:

BreakDemo.java

											
    package com.mycompany;

    public class BreakDemo {

        public static void main(String[] args) {
            int k = 1;
            while (k <= 10) {
                System.out.println(k);
                if (k == 6) {
                    break;
                }
                k++;
            }
            System.out.println("The final value of k is " + k);
        }
    }

output

											
    1
    2
    3
    4
    5
    6
    The final value of k is 6

Η λογική που μόλις υλοποιήσαμε είναι πολύ απλή. Ορίζουμε μια μεταβλητή k με αρχική τιμή 1. Για όσο η τιμή του k παραμένει μικρότερη ή ίση του 10 τότε θα εκτυπώσουμε την τιμή του k. Εάν όμως η τιμή, όπως αλλάζει, γίνει ίση με 6 τότε θέλουμε να βγούμε εντελώς έξω από το loop και να συνεχίσουμε με το υπόλοιπο πρόγραμμα που στην δική μας περίπτωση είναι η εκτύπωση της τελικής τιμής της μεταβλητής k.

Το continue έχει την ιδιότητα όποτε ενεργοποιείται να παραβλέπει την εκτέλεση του loop για κάποια συγκεκριμένη τιμή και να συνεχίσει στο επόμενο βήμα του loop.

Ας δούμε ένα απλό παράδειγμα ενός for loop που χρησιμοποιεί το continue. Δημιουργούμε ένα καινούργιο project με το όνομα ContinueDemo. Μέσα στο project δημιουργούμε ένα πακέτο με το όνομα com.mycompany και μέσα στο πακέτο μια κλάση με το όνομα ContinueDemo. Ο κώδικας της κλάσης είναι ο ακόλουθος:

ContinueDemo.java

											
    package com.mycompany;

public class ContinueDemo {

    public static void main(String[] args) {
        for (int i = 10; i > 0; i--) {
            if (i % 2 == 0) {
                continue;
            }
            System.out.println(i);
        }
    }
}

output

											
    9
    7
    5
    3
    1

Ορίζουμε μια τοπική μεταβλητή i με αρχική τιμή 10. Για όσο το i παραμένει μεγαλύτερο του 0, θα εκτυπώνουμε την τιμή του i. Όμως, αν η τιμή του i διαιρείται ακριβώς με το 2, τότε δεν θέλουμε να εκτυπώσουμε αυτή την τιμή αλλά να προχωρήσουμε και να ελέγξουμε την επόμενη τιμή του i. Τελικό αποτέλεσμα είναι να εκτυπώσουμε μόνο τους μονούς αριθμούς και όχι τους ζυγούς.
